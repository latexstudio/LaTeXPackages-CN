# 常用宏包中文本地化翻译项目-LaTeXStudio发起

## 项目说明
1. 你可以上传您所需的宏包，供大家翻译。 
2. 您可以上传自己翻译好的宏包内容，供大家分享（最好附带源代码，便于以后更新维护）。
3. 您可以申领自己的翻译任务，逐步完善我们的本地化宏包说明库。
